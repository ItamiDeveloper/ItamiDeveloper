### 📖 Resumen Profesional

Desarrollador de Software con sólida experiencia en la construcción de soluciones empresariales. Mi enfoque combina la precisión técnica de la ingeniería con una visión estratégica de negocio, liderando proyectos desde el modelamiento de datos hasta el despliegue en infraestructuras cloud.

* 🏗️ **Arquitectura:** Especialista en Microservicios, API RESTful y Clean Architecture.
* 📱 **Mobile:** Desarrollo de aplicaciones híbridas de alto rendimiento con React Native.
* ☁️ **Cloud & DevOps:** Gestión de instancias EC2, despliegue en AWS y contenedores con Docker.
* 💼 **Consultoría:** Experiencia en modelamiento de negocios y optimización de procesos TI.
